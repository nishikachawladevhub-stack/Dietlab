# Diet Lab Website

A modern, single-page website for Diet Lab - Personalized Nutrition & Online Consultations.

## Features

- 🎨 Sky-blue and white wellness theme
- 📱 Responsive design (mobile-first)
- ✨ Smooth scrolling navigation
- 🎭 Framer Motion animations
- 🏠 Single-page layout with sections: Home, About, Services, Contact

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Project Structure

```
├── components/
│   └── DietLabPage.jsx      # Main page component
├── pages/
│   ├── _app.js             # App wrapper with global styles
│   └── index.js           # Home page
├── styles/
│   ├── globals.css        # Global CSS styles
│   └── DietLabPage.module.css  # Component-specific styles
├── package.json
└── next.config.js
```

## Available Scripts

- `npm run dev` - Start development server on localhost:3000
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Technologies Used

- **Next.js 14** - React framework
- **React 18** - UI library
- **Framer Motion** - Animation library
- **CSS Modules** - Scoped styling

## Navigation

The navbar includes smooth scrolling to:
- **Home** - Hero section
- **About** - About Us section
- **Services** - Services section
- **Contact** - Contact/CTA section

## Customization

All styles are in `styles/DietLabPage.module.css`. The color scheme uses:
- Sky Blue: `#87CEEB`
- Steel Blue: `#4682B4`
- Cadet Blue: `#5F9EA0`
- Light Sky Blue: `#E0F6FF`
